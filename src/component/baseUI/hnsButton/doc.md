# HNS Button 组件

<img src="https://img.shields.io/badge/HNS--Button-v1.0-%23ff6987" alt="HNS-Button">

**可传参数**

| 参数名  |   类型   |        可选         | 必传  |   默认值   |
|:----:|:------:|:-----------------:|:---:|:-------:|
| type | String | primary / default |  否  | default |

**使用示例：**

```jsx
<HNSButton type={"primary"} onClick={handleClick}>按钮</HNSButton>
```

动画和样式 通过切换不同的 `className` 实现不同的效果
